Server-Infos


Standort und Verfügbarkeit


Der Server wird auf einem Rechner in der Schweiz gehostet, der nicht rund um die Uhr laufen kann.
Deshalb ist der Server NICHT 24/7 verfügbar. Er kann jedoch auf Anfrage gestartet werden.
Wenn er nicht mehr benutzt wird, sollte er wieder heruntergefahren werden.
Es ist also wichtig, Bescheid zu geben, wenn der Server nicht mehr benötigt wird.

Server-Spezifikationen


RAM: 48 GB
CPU: i7 13700k
Über WLAN (kann zu erhöhtem Ping führen)
Standort: Schweiz

Mit dem Server verbinden


Der Server kann in Ark: Survival Ascended unter „Inoffiziell“ gefunden werden.
Dabei ist zu beachten, dass die richtigen Sucheinstellungen aktiviert sind,
welche auf dem Bild unten gezeigt werden.

Weitere Informationen


Der Server ist ein Cluster-Server und im Spiel über die Obelisken miteinander verbunden.
Da der Rechner jedoch nicht allmächtig ist, können maximal 4 Welten gleichzeitig laufen.
Wir haben auch überlegt, den Server auf 128 GB RAM zu erweitern,
aber im Moment werden die Welten nur bei Bedarf gewechselt.
